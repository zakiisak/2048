# 2048
2048 made in javascript

Just download and launch the 2048.html file!

Or press the following link to play in your browser:
https://htmlpreview.github.io/?https://github.com/zakiisak/2048/blob/master/2048.html

<img src="/screenshot_1.png?raw=true" width="300" /> <img src="/screenshot_2.png?raw=true" width="300" />

The game is heavily customizable through the URL search parameters.

The parameters available are:

### Grid Size
<code>s=32</code> - sets the grid size to 32x32

### Wave Cell Quanity
<code>wcq=2</code> - how many new cells to spawn after doing an action

### Cell Size
<code>cs=20</code> - Sets the cell size to 20x20 pixels

### Padding
<code>p=2</code> - Sets the padding between the cells to 2 pixels
